## 安装指南

### 方法一：使用官方提供的完整客户端
1. 访问[官方网站](http://eftserver.yzy15.dpdns.org)
2. 点击"立即加入"按钮
3. 根据您的电脑配置选择适合的版本：
   - 3.11版本（还原版）：适合配置较高的电脑，提供最完整的游戏体验
   - 3.9.8版本（低配版）：适合配置较低的电脑，优化性能
4. 按照指引下载并安装游戏客户端

### 方法二：手动安装（进阶用户）
1. **安装基础游戏**
   - 下载并安装《逃离塔科夫》对应版本（0.14或0.16）

2. **安装SPT补丁**
   - 下载对应版本的SPT补丁（3.11或3.9.8）
   - 解压补丁到游戏目录
   - 运行SPT启动器验证安装

3. **安装modSync模组**
   - 下载modSync模组
   - 将模组解压至游戏根目录
   - 启动游戏，modSync会自动同步所需模组

## 配置说明

### 游戏配置
- 启动游戏后，在设置中调整画质以获得最佳体验
- 低配电脑推荐：降低渲染分辨率、纹理质量和阴影效果
- 高配电脑推荐：开启高分辨率纹理和光线追踪效果

### 服务器配置
1. 打开SPT启动器
2. 点击设置图标
3. 开启"DeveloperMode"
4. 输入服务器地址：
   - 3.11版本：`http://server.yzy15.dpdns.org:26969`
   - 3.9.8版本：`http://server.yzy15.dpdns.org:26970`
5. 点击右上角返回按钮
6. 连接服务器并创建账户

## 使用方法

### 首次登录
1. 启动SPT启动器
2. 确保已连接到正确的服务器
3. 输入邮箱创建角色
4. 选择角色版本类型（必须选择Edge Of Darkness、Standard、Left Behind、Prepare To Escape的其中之一）
5. 开始游戏！

### 常见操作
- **服务器状态查询**：在官网首页可以查看服务器是否在线
- **模组更新**：启动游戏时modSync会自动更新模组
- **问题反馈**：加入社区群寻求帮助

## 重要注意事项
- **严禁同时运行线上版和离线版**：这可能导致账号封禁
- **mod更新提示**：如果弹出是否更新mod，请务必选择Continue，否则将会被服务器封禁！
- **修改客户端风险**：不要使用修改过的客户端玩逃离塔科夫线上版
- **尊重社区规则**：遵守服务器规则，保持良好的游戏环境

## 常见问题解答

### Q1: 无法连接到服务器怎么办？
**A1:**
- 确认服务器是否处于开放时间
- 检查网络连接和防火墙设置
- 验证服务器地址是否正确
- 尝试重新启动游戏客户端
- 联系管理员寻求帮助

### Q2: 游戏卡顿严重怎么解决？
**A2:**
- 确保满足最低配置要求（至少16GB内存）
- 降低游戏画质设置
- 关闭其他占用系统资源的程序
- 服务器人流量大时可能出现轻微卡顿，属于正常现象

### Q3: mod无法自动同步怎么办？
**A3:**
- 确保已正确安装modSync模组
- 前往官网资源下载中心下载Backup Mods
- 将下载的模组解压至游戏根目录覆盖
- 重新启动游戏

### Q4: 被服务器封禁了怎么办？
**A4:**
- 检查是否违反了服务器规则
- 通过官方渠道联系管理员申诉
- 详细说明情况并表示歉意

### Q5: 可以使用哪些第三方工具？
**A5:**
- 不允许使用任何作弊或辅助工具
- 不允许使用修改游戏数据的工具
- 仅允许使用官方认可的客户端和模组
