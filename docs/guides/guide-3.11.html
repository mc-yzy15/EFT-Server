3.11版本安装指南 稳定版

稳定运行，经过充分测试，适合大多数玩家

⚠️ 重要警告：注册账户时禁止选择 Unheard（蓝边）账户版本！

服务器严禁在注册时选择 Unheard（蓝边）账户版本！蓝边是账户版本，不是游戏客户端！使用蓝边账户版本将被永久封禁！

1

加入QQ群

首先需要加入我们的QQ群，获取所有安装所需的资源和详细教程。

  1. 请先加入QQ群：1034718897
  2. 进群后请先查看群公告和群精华
  3. 如果有任何问题，可以在群内询问群友
2

下载并安装运行库

在群文件中找到 GRL Package，下载后一键无脑安装。这是补齐所有游戏必备运行库，很多人的电脑缺少运行库所以必须进行这一步。

  1. 进入群文件，找到 GRL Package 文件夹
  2. 下载后运行安装程序
  3. 一键无脑安装，不要改任何东西，直接无脑点下一步安装
  4. 等待安装完成
3

下载游戏本体

在群公告里面找到游戏本体链接，下载对应的3.11版本游戏本体。

  1. 在群公告中找到游戏本体下载链接
  2. 下载3.11版本的SPT-AKI游戏本体
  3. 使用Bandizip、Winrar或7ZIP解压（不要使用Windows系统自带解压功能）
  4. 解压到您选择的目录，确保游戏路径中不包含中文
  5. 确保解压完整，不要有任何遗漏文件
4

配置 Astral 连接器

在群文件中找到"联机必备"文件夹，里面有Astral（稳定连接服务器必备）。

  1. 在群文件中找到"联机必备"文件夹
  2. 下载 Astral 文件夹内的配置链接和启动器
  3. 下载 Astral 软件主程序
  4. 打开 Astral 软件，找到右下角的剪贴板图标
  5. 点击剪贴板图标，会弹出一个框让你粘贴配置链接或房间号
  6. 将配置链接粘贴进去，点击确定
  7. 选择房间：相位剧场（Phase Theater）
  8. 返回 Astral 首页，点击连接
5

运行 Auto Install

在群文件中找到3.11文件夹，下载并运行 Auto Install 自动安装工具。

  1. 在群文件中找到3.11文件夹
  2. 下载 Auto Install 自动安装工具
  3. 运行 Auto Install
  4. 按照提示选择游戏根目录（选择包含 Escape from tarkov.exe 的目录）
  5. 仔细阅读 EULA 最终用户许可协议
  6. 阅读并保证无误之后点击同意
  7. 点击下一步，一直按下一步走完安装流程
  8. 安装完成后会自动弹出一个启动器
6

配置启动器

对自动弹出的启动器进行配置。

  1. 启动器弹出后，点击右上角的 Setting（设置）
  2. 把语言改成中文
  3. 开启底下的 Developer Settings（开发者设置）
  4. 在开发者设置中把 URL 地址改成群文件里提供的 URL
  5. 点击右上角的小箭头返回主界面
7

创建角色并开始游戏

完成上述步骤后，连接服务器并创建角色。

  1. 返回启动器首页后，即可开始创建账号并登录
  2. 输入邮箱创建账号
  3. ⚠️ 服务器无密码验证，账号名称将直接决定他人能否登录你的账号！
  4. ⚠️ 请妥善保管你的账号名称，不要告知他人！
  5. ⚠️ 如果发现登录到他人在账号，请自觉登出，并换一个账号名称！
  6. 选择游戏版本（各版本区别可自行搜索 wiki）：
    • Standard - 对应 Steam 最便宜的版本
    • Left Behind / Prepare To Escape - 不上不下的中间版本
    • Edge Of Darkness - 付费最顶级的体验
  7. ⚠️ 注意：严禁选择 Unheard（蓝边）账户版本！蓝边是账户版本，不是游戏客户端！
  8. 选择完成后点击开始游戏即可进入服务器

常见问题

无法连接到服务器怎么办？

首先确认服务器是否处于开放时间，然后检查您的网络连接和防火墙设置。确保 Astral 已成功连接，URL 地址已正确配置。如果问题依然存在，请尝试联系群管理员寻求帮助。

游戏版本不兼容怎么办？

确保您使用的是 SPT-AKI 3.11 版本，并已正确配置 Astral 连接器和启动器设置。如果仍然无法连接，请检查群公告是否有更新。

游戏过程中出现卡顿怎么办？

确保您的电脑满足最低配置要求，并关闭其他占用系统资源的程序。在服务器人流量较大时，可能会出现轻微卡顿，这是正常现象。

玩离线版有什么注意事项？

1. 不要同时开着线上和离线版一起玩
2. 不要使用修改过的客户端玩逃离塔科夫线上版
3. 如果弹出是否更新mod，请务必选择Continue，否则将会被服务器封禁！！！
