安装指南

按照以下步骤，快速加入我们的服务器

1

安装 SPT3.9.8 版本

首先需要安装《逃离塔科夫》的 SPT3.9.8 版本，这是连接到我们服务器的基础。

安装前准备：

  • 确保您的电脑物理内存达到或超过16GB，虚拟内存设置为物理内存的1-1.2倍
  • 确保游戏路径中不包含中文
  • 建议使用Bandizip、Winrar或7Z解压，避免使用Windows系统自带解压功能

安装步骤：

  1. 下载SPT-AKI 3.9.8客户端（版本0.14.9.1.30626）
  2. 将下载的压缩包解压到您选择的目录
  3. 确保解压完整，不要有任何遗漏文件
  4. 运行SPT.Launcher.exe启动器
2

安装 modSync 模组

安装 modSync 模组以确保您的游戏版本与服务器兼容，并自动同步所需的模组。

安装步骤：

  1. 下载modSync模组
  2. 将模组压缩包解压至游戏根目录
  3. 解压后模组将自动安装到正确位置
  4. 安装完成后模组将自动启用，无需进行手动配置
3

连接到服务器

完成上述步骤后，您就可以连接到我们的服务器了。

连接步骤：

  1. 确保服务器处于开放时间（周末）
  2. 打开SPT.Launcher.exe
  3. 等待出现"不可用"提示后选择"设置"
  4. 开启"DeveloperMode"
  5. 输入服务器地址：http://server.yzy15.dpdns.org:26969
  6. 游戏路径保持不变
  7. 点击右上角小箭头返回主界面
  8. 成功连接后，创建角色并开始游戏

常见问题

无法连接到服务器怎么办？

首先确认服务器是否处于开放时间（周末），然后检查您的网络连接和防火墙设置。如果问题依然存在，请尝试使用游戏加速器或联系管理员寻求帮助。

游戏版本不兼容怎么办？

确保您使用的是 SPT3.9.8 版本（客户端版本0.14.9.1.30626），并已安装 modSync 模组。modSync 会自动同步所需的模组和版本。

游戏中遇到问题如何寻求帮助？

您可以通过页面底部的联系方式联系管理员，或加入我们的 QQ 群（1034718897）寻求社区帮助。

游戏过程中出现卡顿怎么办？

确保您的电脑满足最低配置要求（至少16GB内存），并关闭其他占用系统资源的程序。在服务器人流量较大时，可能会出现轻微卡顿，这是正常现象。

为什么不能使用老版存档？

从SPT3.9.0开始，存档架构已发生变化，老版存档无法直接使用。如果您有编译知识，可以尝试移植存档，否则建议创建新的游戏角色。

玩离线版有什么注意事项？

1. 不要同时开着线上和离线版一起玩
2. 不要使用修改过的客户端玩逃离塔科夫线上版
