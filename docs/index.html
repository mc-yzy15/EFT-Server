安装指南

按照以下步骤轻松安装并连接到我们的服务器，开始您的逃离塔科夫之旅

1

下载游戏客户端

下载 SPT3.9.8 版本的游戏客户端，确保您的游戏版本与服务器兼容。

2

安装 modSync 模组

安装 modSync 模组以确保您的游戏版本与服务器兼容，并自动同步所需的模组。

3

连接到服务器

完成上述步骤后，您就可以连接到我们的服务器了。
